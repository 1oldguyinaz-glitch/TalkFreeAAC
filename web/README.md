# TalkFreeAAC Web

Milestone 2 web foundation.

## What this includes

- Child AAC touchscreen board
- Parent Insights dashboard
- Shared prediction engine
- Local-first browser profile storage
- Austin baseline vocabulary
- Stage-gated predictions
- Speech synthesis
- Responsive layout

## Chromebook / GitHub Codespaces

1. Upload the `web/` folder into the GitHub repo.
2. Open the repo in GitHub Codespaces.
3. In the terminal:

```bash
cd web
npm install
npm run dev
```

4. Open the forwarded browser preview.

## Deploy later

This is ready for Cloudflare Pages or GitHub Pages after the first web test.
