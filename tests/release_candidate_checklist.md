# Release Candidate Manual Checklist

## Repo structure
- release/ folder exists.
- web/app/version.js exists.
- ReleaseReadinessPanel appears in Parent Menu.

## App behavior
- Child AAC board opens.
- Parent Menu opens.
- Emergency profile fields save.
- Backup/restore panel loads.
- Accessibility panel loads.
- AI insights panel loads.
- Team workspace panel loads.

## Before v1.0
- Run engine tests.
- Test offline behavior.
- Confirm privacy policy.
- Add screenshots.
- Build Android wrapper.
