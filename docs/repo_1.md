# repo_1 — TalkFreeAAC Canonical Project Repository

## Mission
TalkFreeAAC is a free, local-first AAC communication system. The user interface stays familiar. The intelligence underneath improves.

## Locked Child Interface
Visible sections:
- Core Needs
- Topics
- Sentence Starters
- Search
- Recents
- Favorites
- Emergency

The child does not access settings, analytics, team tools, or vocabulary management.

## Locked Parent Menu
One locked parent/admin section contains:
- User Profile
- Emergency Description
- Display Settings
- Parent Insights
- Vocabulary Management
- Team Workspace
- Backup/Restore later

## Visual Language
TalkFree Pop:
- Core Needs = deep navy
- Emotions = bright red
- Food & Drinks = orange
- Learning / School = yellow
- Places = green
- People = blue
- Questions = purple
- Actions = lime green
- Safety = dark red
- Time = teal

## Local-First Rule
The AAC communication database stays local unless a parent explicitly enables backup or sharing.

Local data includes:
- Communication history
- Completed sentences
- Tap history
- Word counts
- Recents
- Favorites
- Emergency profile
- Display settings
- Friction logs

Never automatically delete past communication.

## Optional Team Workspace
Team Workspace is opt-in and separate from the local AAC database.

Allowed notes:
- Today I ate ___.
- Today I did well at ___.
- Today I struggled with ___.
- Today we worked on ___.
- Comments.

Teachers/therapists can add notes. They do not control the child's communication board.

## Sentence Engine
When a thought is complete:
1. Speak sentence.
2. Archive sentence locally.
3. Update predictions.
4. Reset board to Sentence Starters.
5. Preserve history.

## Prediction Engine
Predictions use:
- Active context
- Current phrase
- Recent words
- Favorites
- Usage history
- Completed sentences
