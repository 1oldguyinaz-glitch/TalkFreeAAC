# TalkFreeAAC V4 Patch 06 — Adaptive UI + Symbol Framework

## Adds
- ResponsiveGrid component
- SymbolImage component
- TopicRail component
- Layout engine
- Symbol priority resolver
- Adaptive UI CSS

## Replaces
- AACButton now supports professional cartoon symbols with fallback icons
- ChildAAC now uses adaptive grids instead of fixed grids
- App imports adaptive UI styles

## Goal
Make the board scale cleanly across Chromebook, tablet, desktop, and touch displays without normal-use scrolling, while preparing the app for a professional symbol library.
