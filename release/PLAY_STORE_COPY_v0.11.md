# Play Store Copy v0.11

## App name
TalkFreeAAC

## Short description
Free local-first AAC communication with a stable, easy-to-use board.

## Long description
TalkFreeAAC is a free AAC communication app designed for users who need a simple, stable, and predictable way to communicate.

The app includes core needs, topics, sentence starters, search, recents, favorites, emergency communication, local speech output, parent settings, accessibility controls, and optional caregiver/team support.

TalkFreeAAC is local-first by default. Communication data stays on the device unless a parent chooses to export or enable future optional backup features.

## Key features
- Free AAC communication board
- Local-first data storage
- Works offline
- Sentence starters
- Recents and favorites
- Emergency profile
- Parent/Admin controls
- Accessibility settings
- Backup and restore
- Optional team workspace foundation
