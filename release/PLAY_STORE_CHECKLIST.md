# TalkFreeAAC Play Store Checklist

## Required before public release
- Confirm app name: TalkFreeAAC
- Confirm package/app identifier
- Add app icon
- Add feature graphic
- Add screenshots
- Add privacy policy URL
- Confirm local-first data policy
- Confirm no hidden cloud sync
- Confirm child-safe UX
- Confirm parent/admin lock behavior
- Test on Android phone
- Test on Android tablet
- Test offline mode
- Test backup/export
- Test emergency screen
- Test parent menu

## Store positioning
TalkFreeAAC is a free, local-first AAC communication app designed to help users communicate with a stable, predictable interface and optional parent/team support.
