# TalkFreeAAC Store Description Draft

TalkFreeAAC is a free AAC communication app built to keep communication simple, predictable, and accessible.

The app uses a bright, consistent visual language and a stable communication board so users can build familiarity over time. Communication data is stored locally by default, and parent controls are kept separate from the child-facing board.

## Features
- Core Needs board
- Topics
- Sentence Starters
- Search
- Recents
- Favorites
- Emergency communication
- Local speech output
- Parent profile and emergency settings
- Local insights
- Optional team workspace structure
- Backup and restore
- Accessibility settings

TalkFreeAAC is designed for families, caregivers, teachers, and therapists who need a practical AAC tool with a simple, consistent interface.
