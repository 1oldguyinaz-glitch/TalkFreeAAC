# Android Release Checklist

## Before Android build
- Confirm web app runs locally.
- Confirm `npm run build` succeeds.
- Confirm no duplicate misplaced files.
- Confirm child AAC board works.
- Confirm Parent Menu works.
- Confirm Backup/Restore works.
- Confirm Accessibility panel works.
- Confirm local speech output works.

## Android build
- Add Capacitor.
- Add Android platform.
- Open Android Studio.
- Generate debug APK.
- Test on Android phone.
- Test on Android tablet.
- Test offline.

## Play Store
- App icon
- Feature graphic
- Screenshots
- Short description
- Long description
- Privacy policy URL
- Data safety answers
- Content rating
